# Ironclad Clickwrap (ironclad-clickwrap)
Ironclad Clickwrap is a developer-friendly platform for implementing clickwrap agreements, terms of service, and contract acceptance workflows. The Ironclad Clickwrap API enables developers to programmatically manage clickwrap agreements, record user acceptance, and generate audit trails.
